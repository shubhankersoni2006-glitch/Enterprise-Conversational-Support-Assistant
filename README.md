# Enterprise Conversational Support Assistant
A Streamlit-based conversational chatbot that simulates enterprise customer support operations for IT service organizations. The application uses intent-based keyword matching to provide real-time responses for queries related to company services, careers, technical support, and general information.

# Features
Interactive chat interface built with Streamlit
Intent-based query classification
Real-time automated responses
Session state management for chat history
Support for multiple query categories
Lightweight and easy-to-deploy architecture
User-friendly conversational experience
Technologies Used
Python
Streamlit
Natural Language Processing (Rule-Based)
Session State Management
Project Objective
The objective of this project is to demonstrate how conversational AI concepts can be applied to automate customer support interactions in an IT services environment. The chatbot handles common user queries and provides instant responses through an intuitive web interface.

# Supported Query Categories
Company Information
IT Services
Career Opportunities
Technical Support
General Greetings
Frequently Asked Questions
# Project Workflow
User enters a query through the chat interface.
The chatbot processes the input using keyword-based intent matching.
The identified intent is mapped to a predefined response.
The response is displayed instantly to the user.
Chat history is maintained using Streamlit Session State.

Installation and Setup
Clone the Repository
git clone https://github.com/your-username/enterprise-conversational-support-assistant.git
cd enterprise-conversational-support-assistant
Install Dependencies
pip install -r requirements.txt
Run the Application
streamlit run app.py
Open in Browser
After running the command, open the local URL generated by Streamlit (usually http://localhost:8501).
